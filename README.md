# 超级立方豆。	文档正在完善中...



## About
超级立方豆，是一款小巧、炫酷的桌面智能助手， 由疯狂豆Luke 和 助手小A。  
硬件集成: 分光棱镜、红外手势识别、多功能按钮、运动传感器、麦克风、LED灯、喇叭、芯片为 ESP32-Pico-V3-2。  
综合利用硬件资源开发了各种有趣APP应用。
